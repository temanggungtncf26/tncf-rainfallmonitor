## 🚀 Cara Deploy

### Opsi A — GitHub Pages (Gratis, Recommended)

1. Buat repository baru di GitHub (misal: `tncf-rain-monitor`)
2. Upload semua file ke branch `main` atau `gh-pages`
3. Aktifkan GitHub Pages di **Settings → Pages → Source: main branch**
4. Akses via: `https://username.github.io/tncf-rain-monitor/`
5. Buka di Chrome/Android → muncul prompt **"Add to Home Screen"** otomatis ✅

### Opsi B — Deploy ke Server/Hosting

1. Upload semua file ke folder root hosting (cPanel, Nginx, Apache, dll.)
2. Pastikan server menggunakan **HTTPS** (wajib untuk PWA)
3. Pastikan header `Content-Type: application/json` untuk `manifest.json`
4. Pastikan header `Service-Worker-Allowed` diset jika SW berada di subfolder

### Opsi C — Netlify Drop (Paling Mudah)

1. Buka [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag & drop seluruh folder ke browser
3. Netlify otomatis deploy dengan HTTPS ✅

## ⚙️ Konfigurasi GAS (Google Apps Script)

File `index.html` ini dirancang untuk berjalan dalam **2 mode**:

### Mode Simulasi (Standalone / Preview)
- Aktif otomatis saat `google` object tidak tersedia
- Menggunakan data mock internal (mockUsers, mockDB)
- Cocok untuk testing & preview di browser biasa

### Mode GAS (Production)
- Aktif saat dijalankan dalam Google Apps Script (`doGet()`)
- Semua data CRUD dikirim ke Google Sheets via `google.script.run`
- Untuk deploy: upload `index.html` ke GAS project, pastikan nama file `Index` (tanpa ekstensi)

**Catatan GAS:** Karena GAS memiliki sandbox sendiri, file `manifest.json`, `sw.js`, dan `offline.html` **tidak dapat dihosting di GAS**. Untuk PWA penuh di GAS, gunakan pendekatan hosting hybrid:
- Host `index.html` di GAS (sebagai web app utama)
- Host `manifest.json`, `sw.js`, `offline.html` di GitHub Pages / Netlify
- Update URL manifest di `<head>` sesuai hosting

## 🔄 Cache Strategy

| Sumber | Strategi |
|--------|----------|
| App Shell (HTML/icons/manifest) | Cache First |
| CDN (Bootstrap, Chart.js, xlsx) | Stale While Revalidate |
| Google Apps Script API | Network Only |
| Google CDN (logo/gambar) | Network First |

## 🛠️ Update Cache Version

const CACHE_VERSION = 'tncf-v1.3'; // increment version
